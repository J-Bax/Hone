3. Domain Models

All 25 model types grouped by concern. Each is a C# record (or enum) in the Hone.Core.Models namespace.

Experiment Tracking

ExperimentMetadata record

Tracks a single experiment's lifecycle — timing, outcome, branch, metrics, and PR linkage.

+ JsonExtensionData for forward-compatible fields

ExperimentOutcome enum

Discrete result of an experiment comparison.

IterationAttempt record

One attempt within an iterative fix loop — stage, outcome, and diff size.

4. Utilities

Two small static utility classes — zero allocations where possible, source-generated regexes for performance.

JsonUtils

Handles JSON edge-cases produced by k6 and AI agents.

Methods

SanitizeNaN(string json) → string
Replaces NaN, Infinity, and -Infinity with null so the JSON is spec-compliant. Needed because k6 emits non-standard numeric values.

ExtractJsonBlock(string text) → string?
Extracts the content inside ```json … ``` fences. Returns null if no block found. Uses source-generated regex.

ExtractCodeBlock(string text) → string?
Extracts content inside ```csharp … ``` fences. Used to parse AI-generated code responses. Uses source-generated regex.

StringUtils

String manipulation helpers — parity with PowerShell's original Limit-String.

Methods

Truncate(string text, int maxLength) → string
Truncates at a word boundary and appends … (ellipsis). If text.Length <= maxLength, returns the original string unchanged. Mirrors the behavior of the PowerShell Limit-String utility from the original codebase.
