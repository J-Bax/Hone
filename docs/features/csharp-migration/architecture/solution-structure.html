4. Build Infrastructure

Directory.Build.props

  • TreatWarningsAsErrors — all warnings are errors
  • AnalysisLevel = latest-all — maximum analyzer coverage
  • Nullable = enable — nullable reference types enforced

Directory.Packages.props

  • Central package version management
  • Single source of truth for all NuGet versions
  • Projects use <PackageReference> without Version

.editorconfig

  • PascalCase — public members & types
  • _camelCase — private fields (underscore prefix)
  • Allman braces — opening brace on new line

BannedSymbols.txt

  • DateTime → use DateTimeOffset
  • Synchronous File I/O — banned (use async)
  • ArrayList / Hashtable — banned (use generics)

Analyzers

  • Meziantou.Analyzer — best-practice rules
  • Microsoft.VisualStudio.Threading.Analyzers — async/threading
  • Microsoft.CodeAnalysis.BannedApiAnalyzers — enforces BannedSymbols.txt
